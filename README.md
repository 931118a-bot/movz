# Movie Recommendation Project

Django 기반 영화 추천 및 리뷰 웹 서비스입니다.

## 주요 기능
- 회원가입 및 로그인
- 영화 목록 조회 및 검색
- 영화 추천 기능
- 리뷰 등록 / 수정 / 삭제
- 관리자 콘텐츠 관리

## 기술 스택
- Python
- Django
- SQLite
- HTML / CSS

## 프로젝트 구조
- common : 공통 모듈
- config : 설정 파일
- movie : 영화 및 리뷰 로직
- templates : 템플릿 파일
- static : 정적 파일

## 실행 방법
1. 가상환경 생성
2. pip install -r requirements.txt
3. python manage.py migrate
4. python manage.py runserver
